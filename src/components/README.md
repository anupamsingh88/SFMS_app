# Components Directory

This directory contains reusable React Native components.

## Structure
- Place all reusable UI components here
- Each component should ideally have its own folder with:
  - Component file (e.g., `Button.tsx`)
  - Styles file (e.g., `Button.styles.ts`)
  - Tests (if applicable)

## Example
```
components/
├── Button/
│   ├── Button.tsx
│   └── Button.styles.ts
└── Card/
    ├── Card.tsx
    └── Card.styles.ts
```
